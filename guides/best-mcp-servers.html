常见问题

MCP Server有什么用？

MCP Server让AI模型能够调用外部工具和数据源，如文件操作、数据库查询、API调用等，极大扩展了AI的能力边界。

MCP Server用什么语言开发？

官方SDK支持TypeScript和Python，社区有Rust、Go等实现。选择你最熟悉的语言即可。

哪些AI应用支持MCP？

Claude Desktop、Cursor、Windsurf、Cline等主流AI编程工具都已支持MCP协议。
